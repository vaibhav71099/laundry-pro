# laundry-pro
LaundryPro is a modern, fully responsive web application designed to provide users with a seamless laundry booking experience from the comfort of their homes. This project was developed as part of the Tutedude Full Stack Development assignment.

# Key Features

Premium User Interface: A clean, professional, and aesthetic design inspired by modern UI/UX standards.

Dynamic Cart System: Users can select multiple services like Dry Cleaning, Ironing, and Wash & Fold with real-time total price calculation.

Achievements Showcase: Highlights the brand's success with stats like 15+ Services, 240+ Happy Customers, and 2+ Years of Experience.

Automated Email Confirmations: Integrated with EmailJS to send instant booking confirmations to both the customer and the service provider.

Interactive Newsletter: A dedicated section for users to subscribe for latest updates and laundry tips.

Comprehensive Footer: Features essential navigation links, brand identity, and social media connectivity for enhanced user engagement.

# Tech Stack

Frontend: HTML5, CSS3 (Advanced Flexbox & Grid), and Vanilla JavaScript (ES6+).

Icons & Fonts: FontAwesome Icons and Google Fonts (Poppins).

Third-Party API: EmailJS for handling serverless email communications.

Deployment: Hosted live on Netlify / GitHub Pages for easy accessibility.
